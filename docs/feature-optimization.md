- [x] **[Performance]** DONE 2026-06-20 (WS-1) — ✓verified — confirms prior — **3D Bloch &lt;Canvas&gt; runs a 60fps render loop forever (frameloop defaults to 'always')**. BlochSphere3D mounts a react-three-fiber &lt;Canvas&gt; with no frameloop prop, so it defaults to 'always' and re-renders the WebGL scene 60 times a second for the entire life of the component — even after the StateVector lerp has converged on its target and the user is not dragging OrbitControls.
